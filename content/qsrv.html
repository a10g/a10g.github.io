qsrv - A Fortune Server

Return to Launch Page

qsrv is a simple Flask application serving fortunes via simple REST API.
It reads from a directory of BSD-style fortune files, without the strfile(8) requirement.

An implementation is deployed at http://qsrv.a10g.com/.
